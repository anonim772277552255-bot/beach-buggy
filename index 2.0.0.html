НАСТРОЙКА БОТА

1Создай бота в Telegram

  1. Открой Telegram и найди @BotFather
  2. Отправь команду /newbot
  3. Придумай имя боту (например: BeachBuggyChat)
  4. Придумай username (например: beach_buggy_chat_bot)

2Введи токен бота

BotFather прислал токен. Вставь его сюда:

Пример: 123456789:ABCdefGHIjklMNOpqrSTUvwxyz

3Создай группу для чата

  1. Создай новую группу в Telegram
  2. Добавь своего бота в группу
  3. Сделай бота администратором
  4. Отправь в группу любое сообщение
  5. Нажми кнопку ниже для получения ID группы
